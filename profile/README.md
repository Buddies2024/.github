## 1. 서비스 소개
스프링은 교환일기를 통해 일상을 기록하고 추억을 나누는 커뮤니티 서비스입니다.<br>
교환일기는 여러 사람이 한 권의 일기장을 돌려가며 각자의 이야기를 공유하는 방식으로,<br>
스프링은 디지털 환경에서도 이 따뜻함을 이어가고자 하는 목적에서 시작되었습니다.<br>
공책의 스프링이 여러 장을 묶어주듯, 함께하는 사람들을 이어주는 연결고리가 되기를 기대합니다.
